# Recipe Bay 🍳

Recipe Bay is a modern, user-friendly platform designed to connect food enthusiasts and share culinary experiences. Whether you're a professional chef or a home cook, Recipe Bay provides the perfect space to discover, share, and celebrate the joy of cooking.

## 🌟 Features

- **Recipe Discovery**: Browse through a vast collection of recipes from various cuisines
- **User Profiles**: Create your personal profile and share your culinary journey
- **Recipe Sharing**: Upload and share your favorite recipes with the community
- **Category Filtering**: Easily find recipes by cuisine type or dietary preferences
- **Interactive UI**: Modern, responsive interface with intuitive navigation
- **Community Engagement**: Like, save, and comment on recipes

## 🚀 Technologies Used

- **Frontend Framework**: React 18 with TypeScript
- **Styling**: Tailwind CSS for modern, responsive design
- **Icons**: Lucide React for beautiful, scalable icons
- **Build Tool**: Vite for fast development and optimized builds
- **Type Checking**: TypeScript for enhanced code reliability
- **Code Quality**: ESLint for code consistency

## 🛠️ Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Beloveyeboah/Recipe-Bay.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Recipe-Bay
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```
